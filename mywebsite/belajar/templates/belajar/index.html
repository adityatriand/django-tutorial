Cara Install Django

  1. Install Python

    2. Anda bisa download python versi terbaru di website resminya yaitu https://www.python.org/downloads.Selanjutnya, pilih file installer dan sesuaikan dengan sistem komputer Anda.

  2. Cek Python Version

    3. Untuk melihat apakah instalasi python Anda berhasil atau tidak, Anda bisa melakukan pengecekan versi python Anda, dengan cara membuka command prompt pada windows lalu ketik: python --version

  3. Upgrade pip

    4. Dengan mengetik command berikut: python -m pip install --upgrade pip

  4. Membuat virtual environment

    5. python -m venv env

  5. Mengaktifkan virtual environment

    6. env\Scripts\activate

  6. Install django

    7. pip install django atau jika Anda ingin install spesifik version, Anda bisa mengetik command berikut: pip install django==3.2.4

  7. Cek Django Version

    8. Jika instalasi sudah berhasil, Anda bisa melakukan pengecekan versi django Anda, dengan cara mengetik command berikut: django-admin --version

  8. Start new project

    9. django-admin startproject nama_website

  9. Run the server

    10. menggunakan command berikut: python manage.py runserver
